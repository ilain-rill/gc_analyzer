# gc_analyzer
Функциональный расчет и анализ GC-состава генома скользящим окном
программа рассчитывает частоту встречаемости определённых нуклеотидов ( GC-пары — гуанин и цитозин) по длине последовательности. Окно заданной длины движется с заданным шагом , и на каждой позиции программа вычисляет процент GC в этом фрагменте. На выходе получается график распределения GC-состава вдоль гена или генома — это нужно для поиска CpG-островков (участков, богатых GC), которые важны для регуляции работы генов.
